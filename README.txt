
This version of the game is being ran on linux mint 17 with 

Python 2.7.6

and 

pygame '1.9.1release'

note:

instructions for how to look up the pygame version can be found 
here: 

http://www.opensourceforu.com/2011/05/get-started-with-pygame/

basically, run the following commands (in the order that they are presented): 

python
(this will bring up a prompt)
pygame.ver
(this will spit out the version number)

done!
